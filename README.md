### Prerequisites

- [Node.js](https://nodejs.org/) v14+
- [Git](https://git-scm.com/)

### Local Setup

```bash
# 1. Clone the repository
git clone https://github.com/akash4550/mahadbt-awareness-portal.git

# 2. Navigate into the project
```
